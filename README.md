# Sistema-Simples-de-de-Controle-de-Estoque
Desenvolvimento de um sistema de gerenciamento de estoque. O sistema realizava funcionalidades como login, cadastro de usuários e funcionários, cadastro de produtos, controle de estoque, controle de solicitações e geração de relatórios. 

O sistema foi feito com a participação de 3 Integrantes, contando comigo.
